# narashi 連鎖暴走(over-merge)検証用 ディストラクタ集 (テストデータ)
#
# 全行が「単独語」= どれとも統合されてはいけない負例。glossary.txt とは語が
# 重複しない(重複するとラベルが壊れる。src/eval.rs のテストで保証)。
#
# 目的: 単連結クラスタリング(union-find による推移閉包)が、少数の「ハブ語」を
# 介して無関係なクラスタを 1 つの巨大ゴミクラスタへ連鎖させてしまう「連鎖暴走」を
# 再現・検知するためのストレスデータ。default_glossary() と extended() で連結し、
# largest_cluster_ratio / groups_in_largest が暴走を捉えるかを測る。
#
# 実データのタグ群に多い特徴を意図的に含む:
#   (A) ハブ語: 数字・単一文字・短い略語など、埋め込みが中心付近に潰れて何にでも
#       中程度に近くなり、無関係なクラスタ間の「橋」になりやすい汎用短トークン。
#   (B) 無関係プール: 多ドメインから互いに非同義の実在風タグを多数。本来は大半が
#       単独のまま残るべきで、ここが巨大クラスタに飲み込まれたら暴走。

# ============================================================
# (A) ハブ語 — 汎用短トークン(連鎖暴走の橋になりやすい)
# ============================================================
