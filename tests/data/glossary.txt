# narashi 評価用 用語集 (テストデータ)
#
# モデルを差し替えるたびに同じ物差しで精度を比較するための「正解データ」です。
# 1 行 = 1 つの同義語グループ。メンバはカンマ (,) 区切り。
#   - 同じグループ内のペア  => 統合されるべき (正例)
#   - 異なるグループ間のペア => 統合されるべきでない (負例)
# メンバが 1 つだけの行は「単独語 (ディストラクタ)」で、他のどれとも統合されない。
# `#` 始まりの行と空行は無視されます。用語は自由に追記してください。
#
# narashi は多言語埋め込みを使うため、実運用では日本語・中国語・英語が
# 混在しがちです。各グループは表記ゆれ (かな/カナ/漢字/送り仮名) に加えて、
# 同義の外来語・英語・中国語も同じグループにまとめ、言語をまたぐ統合能力も測ります。
